## 二、任务内容

### 2.1 文档结构

文档应包含以下章节：

1. **标题**：ADR-003: 选择 FPM383C 作为指纹传感器
2. **状态**：已接受（Accepted）
3. **背景**：
   - 项目需求（FIDO2 生物识别、本地比对、安全要求）
   - 技术约束（FAR < 0.001%、FRR < 3%、匹配时间 < 1秒）
   - 成本约束（指纹传感器成本 < $15）
   - 决策触发（需确定指纹传感器以开始硬件设计）
4. **决策**：
   - 最终决策：FPM383C 电容式指纹模块
   - 决策理由（价格适中、内置算法、小体积、UART 接口简单）
5. **考虑的选项**：
   - FPM383C vs GT521F32 vs R306 vs AS608 对比分析
   - 详细技术参数对比表
   - 成本对比
   - 性能对比（FAR、FRR、匹配时间）
   - 安全性对比
6. **技术规格**：
   - 类型：电容式按压传感器
   - 分辨率：192x192 像素
   - 采集面积：10x10 mm
   - 模板容量：200 个指纹模板
   - 误识率（FAR）：< 0.001%
   - 拒识率（FRR）：< 2%
   - 匹配时间：< 1 秒
   - 接口：UART (9600-115200 bps)
   - 工作电压：3.3V
   - 工作电流：~20-50mA
   - 待机电流：< 5mA
7. **成本分析**：
   - 单价：$10-15
   - 采购渠道建议
   - 模块 vs 芯片采购对比
8. **安全性评估**：
   - 模板存储安全性
   - 本地比对安全性
   - 防假指纹能力
   - 模板加密机制
9. **性能评估**：
   - FAR/FRR 实测数据
   - 匹配速度实测
   - 不同环境性能表现
10. **风险评估**：
    - SDK 兼容性风险
    - 模板容量限制风险
    - 供应链风险
    - 性能不达标风险
11. **后果**：
    - 正面影响（成本适中、开发简单、小体积）
    - 负面影响（模板容量有限、无活体检测）
    - 后续决策依赖（LLD 设计、PCB 设计）
12. **替代方案分析**：
    - GT521F32 方案详细分析（高性能）
    - R306 方案详细分析（低成本光学式）
    - AS608 方案详细分析（国产方案）
13. **实施计划**：
    - 硬件采购建议
    - SDK 集成方案
    - 验证里程碑
14. **备注**：
    - 参考资料
    - 更新记录
