## 二、任务内容

### 2.1 文档结构

文档应包含以下章节：

1. **概述**：
   - ATECC608A 简介
   - 核心特性
   - 适用场景
   - 文档目的
2. **安全等级规格**：
   - 安全等级：EAL4+（Common Criteria）
   - 安全特性：
     - 防篡改检测
     - 防侧信道攻击
     - 密钥不可导出
     - 安全启动支持
   - 物理安全：
     - 防电压攻击
     - 防温度攻击
     - 防光照攻击
3. **密钥存储规格**：
   - 密钥槽位：16 个独立槽位
   - 槽位类型：
     - Slot 0-9：密钥存储（ES256）
     - Slot 10-13：数据存储
     - Slot 14：attestation 密钥
     - Slot 15：ECDH 密钥
   - 槽位容量：每槽位最多 1 个密钥
   - 密钥类型：ES256（ECDSA P-256）
   - 密钥权限：
     - 不可读取
     - 可签名
     - 可生成
     - 可删除（覆盖）
4. **算法支持规格**：
   - ECDSA（ES256）：
     - 曲线：NIST P-256（secp256r1）
     - 签名：ECDSA with SHA-256
     - 公钥格式：uncompressed（65 bytes）
     - 签名格式：raw signature（64 bytes）
   - ECDH：
     - 曲线：NIST P-256
     - 密钥协商：ECDH
     - 输出：shared secret（32 bytes）
   - SHA-256：
     - 输入：任意长度
     - 输出：32 bytes
   - AES-128（可选）：
     - 加密模式：CBC
     - 密钥长度：128 bits
5. **接口规格**：
   - 接口类型：I2C
   - I2C 地址：0xC0（7-bit: 0x60）
   - I2C 速度：最高 1 MHz（Fast Mode Plus）
   - 数据格式：I2C 标准
   - 响应时间：< 50ms
6. **功能规格**：
   - 密钥生成：GenKey 命令
   - 签名操作：Sign 命令
   - 验证签名：Verify 命令
   - 密钥协商：ECDH 命令
   - 哨希：SHA 命令
   - 随机数生成：Random 命令
   - 读写数据：Read/Write 命令
   - 槽位配置：Lock/Write 配置
7. **功耗规格**：
   - 工作电压：2.0-5.5V
   - 工作电流：5-10mA（签名时）
   - 待机电流：< 1mA
   - 休眠电流：< 100µA（Sleep 模式）
8. **物理规格**：
   - 封装：8-pad SOIC
   - 尺寸：5x6x1.5 mm
   - 工作温度：-40°C to +85°C
   - 存储温度：-40°C to +85°C
   - ESD 保护：HBM > 2000V
9. **引脚定义**：
   - 引脚图
   - 引脚功能表（VCC、GND、SDA、SCL、RST、NC）
   - 引脚电气特性
10. **通信协议**：
    - I2C 命令格式：
      - 包头：特定字节
      - 命令码：1 byte
      - 参数：N bytes
      - 数据：N bytes
      - CRC：2 bytes
    - 常用命令：
      - GenKey：0x40
      - Sign：0x41
      - Verify：0x45
      - ECDH：0x43
      - SHA：0x47
      - Random：0x1B
      - Read：0x02
      - Write：0x12
    - 响应格式：
      - 状态码：1 byte（成功=0x00、失败=其他）
      - 数据：N bytes
      - CRC：2 bytes
11. **电气特性**：
    - 最大电压：5.5V
    - 最大电流：15mA
    - 输入/输出特性
    - 电气参数表
12. **密钥槽位分配建议**：
    - IBio 项目密钥槽位分配方案：
      - Slot 0-3：凭证密钥（4 个 resident credential）
      - Slot 4-9：扩展凭证密钥（6 个备用）
      - Slot 14：attestation 密钥
      - Slot 15：PIN 加密密钥（ECDH）
    - 槽位权限配置建议
13. **应用建议**：
    - 电路设计建议
    - I2C 配置建议
    - CryptoAuthLib 集成建议
    - 密钥管理建议
    - 安全配置建议
14. **参考资料**：
    - 数据手册链接
    - CryptoAuthLib 链接
    - 应用笔记链接
