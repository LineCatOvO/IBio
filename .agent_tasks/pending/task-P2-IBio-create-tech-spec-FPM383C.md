## 二、任务内容

### 2.1 文档结构

文档应包含以下章节：

1. **概述**：
   - FPM383C 简介
   - 核心特性
   - 适用场景
   - 文档目的
2. **传感器规格**：
   - 类型：电容式按压传感器
   - 分辨率：192x192 像素
   - 采集面积：10x10 mm
   - 像素密度：192 dpi
   - 图像深度：8-bit 灰度
   - 采集方式：按压式
3. **性能规格**：
   - 误识率（FAR）：< 0.001% (1/100,000)
   - 拒识率（FRR）：< 2%
   - 匹配时间：< 1 秒
   - 注册时间：< 3 秒（多次采集）
   - 模板容量：200 个指纹模板
   - 注册次数：建议 3-5 次采集
   - 识别模式：1:1 比对、1:N 搜索
4. **接口规格**：
   - 接口类型：UART
   - 波特率：9600-115200 bps（默认 9600）
   - 数据格式：8N1（8 数据位、无校验、1 停止位）
   - 协议：自定义 UART 协议
   - 响应时间：< 500ms
5. **指纹算法规格**：
   - 算法类型：内置指纹算法（模块内置）
   - 特征提取：minutiae-based
   - 比对算法：模板匹配
   - 模板格式：模块内部格式（不可导出）
   - 模板存储：模块内部 Flash
   - 模板加密：模块内部加密（可选）
6. **功耗规格**：
   - 工作电压：3.3V (2.8-3.6V)
   - 工作电流：20-50mA（采集时）
   - 待机电流：< 5mA
   - 休眠电流：< 1mA（可选）
7. **物理规格**：
   - 尺寸：20x20x5 mm（模块尺寸）
   - 传感器尺寸：10x10 mm
   - 连接器：6-pin 排针（UART + 电源）
   - 工作温度：-20°C to +60°C
   - 存储温度：-40°C to +85°C
   - ESD 保护：HBM > 2000V
8. **引脚定义**：
   - 引脚图
   - 引脚功能表（VCC、GND、TX、RX、EN、TOUCH）
   - 引脚电气特性
9. **通信协议**：
   - UART 包格式：
     - 包头：0xEF01
     - 地址：4 bytes
     - 包标识：1 byte
     - 包长度：2 bytes
     - 命令码：1 byte
     - 数据：N bytes
     - 校验：2 bytes
   - 常用命令：
     - 采集指令：0x01
     - 比对指令：0x03
     - 注册指令：0x05
     - 删除指令：0x0C
     - 查询指令：0x1D
   - 响应格式：
     - 确认码：1 byte（成功=0x00、失败=0x01等）
     - 数据：N bytes
10. **电气特性**：
    - 最大电压：3.6V
    - 最大电流：100mA
    - 输入/输出特性
    - 电气参数表
11. **应用建议**：
    - 电路设计建议
    - UART 配置建议
    - 指纹采集建议
    - 模板管理建议
    - 性能优化建议
12. **参考资料**：
    - 技术文档链接
    - SDK 链接
    - 应用笔记链接
