## 二、任务内容

### 2.1 文档结构

文档应包含以下章节：

1. **标题**：ADR-006: 选择 USB 供电 + LDO 电源管理方案
2. **状态**：已接受（Accepted）
3. **背景**：
   - 项目需求（USB 供电、功耗控制）
   - 技术约束（USB VBUS 5V、各模块电压需求）
   - 成本约束（电源管理成本最小化）
   - 功耗需求（待机 < 10mA、工作 < 100mA）
   - 决策触发（需确定电源管理方案以满足功耗要求）
4. **决策**：
   - 最终决策：USB VBUS 供电 + LDO/DCDC 降压至 3.3V
   - 决策理由（USB 供电无电池成本、LDO 简单可靠、成本最低）
5. **考虑的选项**：
   - USB 供电 vs 电池供电 vs 双供电对比分析
   - LDO vs DCDC vs 开关电源对比
   - 成本对比
   - 功耗对比
6. **技术规格**：
   - 输入电压：USB VBUS 5V
   - 输出电压：3.3V
   - 输出电流：最大 500mA（满足所有模块需求）
   - LDO 型号建议：AMS1117-3.3 / XC6206P332
   - 效率：LDO ~60%（5V→3.3V），DCDC ~85%
   - 静态电流：LDO < 10mA，DCDC < 5mA
7. **成本分析**：
   - USB 供电方案成本：$0（无电池）
   - LDO 成本：$0.5-1
   - DCDC 成本：$1-2
   - 电池供电方案成本：$5-10（电池 + 充电管理）
   - 成本对比表
8. **功耗分析**：
   - 各模块功耗需求：
     - ESP32-S3：50-100mA（工作），<10mA（待机）
     - FPM383C：20-50mA（工作），<5mA（待机）
     - ATECC608A：5-10mA（工作），<1mA（待机）
     - LED：10-20mA（工作）
   - 总功耗估算：
     - 待机状态：< 10mA
     - 指纹采集：~80mA
     - 签名操作：~60mA
     - BLE 工作：~50mA
9. **电源架构设计**：
   - USB-C VBUS → LDO/DCDC → 3.3V → 各模块
   - 电源架构图
   - 各模块电源分配
   - 稳压器选型建议
10. **风险评估**：
    - USB 供电依赖风险
    - 功耗超标风险
    - LDO 发热风险
    - 电源稳定性风险
11. **后果**：
    - 正面影响（成本最低、无电池、简单可靠）
    - 负面影响（仅 USB 供电、无法移动使用）
    - 后续决策依赖（PCB 设计、功耗优化）
12. **替代方案分析**：
    - 电池供电方案详细分析（移动使用）
    - 双供电方案详细分析（USB + 电池）
    - DCDC 方案详细分析（高效率）
13. **实施计划**：
    - 稳压器选型
    - 电源测试方案
    - 功耗优化方案
    - 验证里程碑
14. **备注**：
    - 参考资料
    - 更新记录
