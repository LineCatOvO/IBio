# ADR-003: 选择 FPM383C 作为指纹传感器

**状态**：已接受（Accepted）

**决策日期**：2026-04-06

**决策者**：IBio 项目团队

---

## 一、标题

选择 FPM383C 电容式指纹模块作为 IBio FIDO2 安全密钥的指纹传感器

---

## 二、背景

### 2.1 项目需求

IBio 项目是一个 DIY FIDO2 安全密钥，需要实现生物识别（指纹）验证功能。FIDO2 标准对生物识别认证器有明确要求：

| 需求项 | FIDO2 要求 | 项目目标 |
|--------|------------|----------|
| **误识率（FAR）** | < 0.001% (1/100,000) | < 0.001% |
| **拒识率（FRR）** | < 3% | < 2% |
| **匹配时间** | < 2 秒 | < 1 秒 |
| **本地比对** | 模板不可导出，本地比对 | 必须 |
| **模板容量** | 每用户至少 5 个指纹 | 最多 200 个 |

### 2.2 技术约束

| 约束条件 | 说明 |
|----------|------|
| **接口约束** | 必须支持 UART/I2C/SPI 接口，便于与 ESP32-S3 集成 |
| **功耗约束** | 工作电流 < 100mA，待机电流 < 10mA |
| **体积约束** | 适合便携设备，采集面积 < 15x15 mm |
| **安全约束** | 模板存储必须安全，禁止模板导出 |
| **成本约束** | 指纹传感器成本 < $15 |

### 2.3 成本约束

项目总成本目标为 < $31，指纹传感器作为核心组件，预算分配如下：

| 项目 | 预算（预估） |
|------|--------------|
| **ESP32-S3 主控** | $5-8 |
| **指纹传感器** | $10-15 |
| **ATECC608A 安全芯片** | $1-2 |
| **USB-C 连接器 + PCB** | $5-8 |
| **总计** | $21-31 |

### 2.4 决策触发

需要确定指纹传感器型号，以便：
1. 完成 PCB 布局设计
2. 确定 GPIO/UART 接口配置
3. 采购硬件原型验证组件
4. 制定 SDK 集成方案

---

## 三、决策

### 3.1 最终决策

选择 **FPM383C 电容式指纹模块** 作为 IBio 项目的指纹传感器。

| 项目 | 选择 |
|------|------|
| **传感器型号** | FPM383C |
| **传感器类型** | 电容式按压传感器 |
| **厂商** | Fingerprint Cards (FPC) |
| **接口** | UART (9600-115200 bps) |
| **价格** | $10-15 |

### 3.2 决策理由

选择 FPM383C 的核心理由：

| 理由 | 说明 |
|------|------|
| **价格适中** | $10-15 符合成本目标，相比 GT521F32 ($15-20) 更低 |
| **内置算法** | 内置指纹采集、特征提取、比对算法，减少开发工作量 |
| **小体积** | 采集面积 10x10 mm，适合便携设备 |
| **UART 接口简单** | 标准 UART 接口，ESP32-S3 集成简单，无需复杂驱动 |
| **性能满足** | FAR < 0.001%、FRR < 2%、匹配时间 < 1秒，满足 FIDO2 要求 |
| **模板安全** | 模板存储在模块内部，禁止导出，满足本地比对要求 |

**评估结论**：本地比对安全，满足 FIDO2 要求。

| **成本适中** | $10-15 符合项目预算，成本可控 |
| **开发简单** | UART 接口集成简单，SDK 完善 |
| **体积小** | 10x10 mm 采集面积，适合便携设备 |
| **性能满足** | FAR/FRR 满足 FIDO2 标准 |
| **本地比对** | 模板安全，满足 FIDO2 要求 |

### 10.2 负面影响

| 影响 | 说明 | 缓解措施 |
|------|------|----------|
| **模板容量有限** | 200 个模板，多用户受限 | 单用户设计 |
| **无活体检测** | 标准版无活体检测 | 用户教育、可选升级 FPM383F |
| **FRR 略高** | 2% FRR，略高于 GT521F32 | 优化采集流程、多次采集 |
| **供应链依赖** | FPC 模块依赖单一厂商 | 多渠道采购、备选方案 |

### 10.3 后续决策依赖

## 十一、替代方案分析

### 11.1 替代方案一：GT521F32（高性能方案）

**方案描述**：
- 传感器：GT521F32 电容式
- 特点：高性能、大模板容量

**对比分析**：

| 项目 | GT521F32 | FPM383C |
|------|----------|---------|
| **成本** | $15-20（较高） | $10-15（适中） |
| **FRR** | < 1%（更优） | < 2%（满足） |
| **模板容量** | 1000+（更大） | 200（足够） |
| **体积** | 14x11 mm（较大） | 10x10 mm（较小） |
| **活体检测** | 有（更安全） | 无 |

**适用场景**：高安全要求、多用户场景、成本不敏感

**未选择原因**：
1. 价格较高，超出预算
2. 体积较大，不适合便携设备
3. 单用户场景模板容量过剩

### 11.2 替代方案二：R306（低成本光学方案）

**方案描述**：
- 传感器：R306 光学式
- 特点：低成本、光学成像稳定

**对比分析**：

| 项目 | R306 | FPM383C |
|------|------|---------|
| **成本** | $10-15（相同） | $10-15（相同） |
| **类型** | 光学式 | 电容式 |
| **体积** | 16x21 mm（大） | 10x10 mm（小） |
| **功耗** | 100-200mA（高） | 20-50mA（低） |
| **电压** | 5V（需转换） | 3.3V（兼容） |
| **光照影响** | 有影响 | 无影响 |

**适用场景**：低成本、体积不敏感、固定安装场景

**未选择原因**：
1. 体积大，不适合便携设备
2. 功耗高，增加电源设计复杂度
3. 需要 5V 供电，与 ESP32-S3 3.3V 不兼容
4. 光学式可能受光照影响

### 11.3 替代方案三：AS608（国产方案）

**方案描述**：
- 传感器：AS608 电容式
- 特点：国产、价格低、中文资料

**对比分析**：

| 项目 | AS608 | FPM383C |
|------|-------|---------|
| **成本** | $8-12（更低） | $10-15（适中） |
| **厂商** | 合泰（国产） | FPC（国际） |
| **文档** | 中文为主 | 英文为主 |
| **社区资源** | 较少 | 中等 |
| **模板容量** | 300 | 200 |
| **FRR** | < 1%（更优） | < 2%（满足） |

**适用场景**：国产方案、成本敏感、中文文档需求

**未选择原因**：
1. 社区资源少，开发支持有限
2. 国际厂商资料更完善
3. FPM383C 在 FIDO2 开源项目中有参考（SoloKeys）
