# ADR-006: 选择 USB 供电 + LDO 电源管理方案

**状态**：已接受（Accepted）

**决策日期**：2026-04-06

**决策者**：IBio 项目团队

---

## 一、标题

选择 USB VBUS 供电 + LDO 降压电源管理方案实现 IBio FIDO2 安全密钥

---

## 二、背景

### 2.1 项目需求

IBio 项目是一个 DIY FIDO2 安全密钥，电源管理需求如下：

1. **供电方式**：USB-C VBUS 供电（5V 输入）
2. **功耗控制**：待机功耗低，满足 USB 规范
3. **稳定性**：电源稳定，无电压波动
4. **成本控制**：电源管理成本最小化

### 2.2 技术约束

| 约束条件 | 说明 |
|----------|------|
| **输入电压** | USB VBUS 5V（4.75V-5.25V 范围） |
| **输出电压** | 3.3V（ESP32-S3、FPM383C、ATECC608A 工作电压） |
| **输出电流** | 最大 500mA（满足所有模块峰值需求） |
| **功耗规范** | 符合 USB 2.0 规范（单端口最大 500mA） |
| **成本约束** | 电源管理成本最小化 |
| **体积约束** | PCB 空间有限，电源模块需小型化 |

### 2.3 功耗需求

| 状态 | 功耗需求 | 说明 |
|------|----------|------|
| **待机状态** | < 10mA | 设备空闲，等待用户操作 |
| **指纹采集** | ~80mA | FPM383C 采集 + ESP32-S3 处理 |
| **签名操作** | ~60mA | ATECC608A 签名 + ESP32-S3 处理 |
| **BLE 工作** | ~50mA | BLE 广播/连接 |
| **峰值功耗** | < 100mA | 所有模块同时工作 |

### 2.4 决策触发

需要确定电源管理方案：
- 选择供电方式（USB 供电 vs 电池供电 vs 双供电）
- 选择降压方案（LDO vs DCDC vs 开关电源）
- 确定稳压器型号和规格
- 设计电源架构和分配方案

---

## 三、决策

### 3.1 最终决策

选择以下电源管理方案：

| 项目 | 选择 | 具体方案 |
|------|------|----------|
| **供电方式** | USB VBUS 供电 | USB-C 接口 5V 输入 |
| **降压方案** | LDO 稳压器 | AMS1117-3.3 或 XC6206P332 |
| **输出电压** | 3.3V | 所有模块统一供电 |
| **输出电流** | 500mA | 满足峰值需求 |
| **电源架构** | 单电源架构 | USB VBUS → LDO → 3.3V → 各模块 |

### 3.2 决策理由

选择该方案的核心理由：

1. **USB 供电无电池成本**：USB 供电无需电池、充电管理芯片，成本最低
2. **LDO 简单可靠**：LDO 稳压器结构简单，无开关噪声，输出稳定
3. **成本最低**：电源管理总成本 < $1
4. **设计简单**：电源电路简单，PCB 设计容易
5. **符合 DIY 定位**：无电池维护，用户无需担心充电

## 四、考虑的选项

### 4.1 供电方式选项对比

| 选项 | 优点 | 缺点 | 成本 |
|------|------|------|------|
| **USB 供电** | 无电池成本、设计简单、随时可用 | 仅 USB 供电、无法移动使用 | $0 |
| **电池供电** | 移动使用、独立供电 | 电池成本高、需充电管理、体积大 | $5-10 |
| **双供电** | USB + 电池、灵活使用 | 成本最高、设计复杂 | $6-12 |

**对比结论**：
- USB 供电成本最低，适合 DIY 项目
- 电池供电适合移动场景，但成本高
- 双供电最灵活，但成本和复杂度最高

**选择 USB 供电理由**：
1. 成本最低（无电池成本）
2. 设计最简单（无充电管理）
3. 符合 DIY 定位（安全密钥主要用于 PC 连接）
4. 无电池维护（用户无需担心充电）

### 4.2 降压方案选项对比

| 选项 | 优点 | 缺点 | 成本 | 效率 |
|------|------|------|------|------|
| **LDO 稳压器** | 简单、无噪声、稳定、低成本 | 效率低（~60%）、发热 | $0.5-1 | ~60% |
| **DCDC 降压** | 效率高（~85%）、低发热 | 开关噪声、成本略高 | $1-2 | ~85% |
| **开关电源模块** | 集成度高、效率高 | 成本高、体积大 | $2-3 | ~90% |

**对比结论**：
- LDO 成本最低，无噪声，适合对噪声敏感的应用
- DCDC 效率高，适合功耗高、发热敏感的应用
- 开关电源模块集成度高，但成本高

**选择 LDO 稳压器理由**：
1. 成本最低（$0.5-1）
2. 无开关噪声（安全芯片和指纹模块对噪声敏感）
3. 输出稳定（无电压波动）
4. 设计简单（无复杂电路）
5. 功耗足够（总功耗 < 100mA，发热可控）

### 4.3 LDO 型号对比

| 型号 | 输入电压 | 输出电压 | 输出电流 | 静态电流 | 成本 | 特点 |
|------|----------|----------|----------|----------|------|------|
| **AMS1117-3.3** | 4.5-15V | 3.3V | 1A | ~10mA | $0.5-1 | 通用、成熟、低成本 |
| **XC6206P332** | 1.7-6V | 3.3V | 200mA | <1μA | $0.5-1 | 低静态电流、小体积 |
| **MCP1700-3302E** | 2.3-6V | 3.3V | 250mA | <2μA | $0.5-1 | Microchip、低功耗 |
| **SPX3819M5-L-3-3** | 2.5-16V | 3.3V | 500mA | ~1mA | $0.5-1 | 高输出电流 |

**对比结论**：
- AMS1117-3.3 输出电流最大（1A），通用成熟
- XC6206P332 静态电流最低（<1μA），适合低功耗
- MCP1700 输出电流适中，静态电流低
- SPX3819 输出电流适中，成本适中

**选择 AMS1117-3.3 理由**：
1. 输出电流最大（1A），满足峰值需求
2. 成本最低（$0.5-1）
3. 成熟稳定，广泛使用
4. 封装选择多（SOT-223、TO-252）
5. 容易采购

### 9.2 LDO 发热风险

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|----------|
| **LDO 过热** | 低 | 中 | 功耗 < 100mA，发热可控 |

- 输入电压：5V
- 输出电压：3.3V
- 压差：1.7V
- 最大电流：100mA（实际峰值）

- AMS1117 热阻（SOT-223）：~80°C/W
- 温升：170mW × 80°C/W = 13.6°C
- 室温 25°C 时，LDO 温度：38.6°C（安全）

## 十一、替代方案分析

### 11.1 替代方案一：USB 供电 + DCDC 降压

**方案描述**：
- 供电方式：USB VBUS 5V
- 降压方案：DCDC 开关稳压器
- 稳压器型号：TPS562201 或 MP1584
- 效率：~85%

**对比分析**：

| 项目 | USB + DCDC | USB + LDO |
|------|------------|-----------|
| **效率** | ~85%（高） | ~66%（低） |
| **发热** | 低 | 中 |
| **开关噪声** | 有 | 无 |
| **成本** | $1.7-2.7 | $0.6-1.1 |
| **设计复杂度** | 中 | 低 |
| **适用性** | 功耗高场景 | 功耗低场景 |

**未选择原因**：
1. 成本较高（$1.7-2.7 vs $0.6-1.1）
2. 开关噪声可能影响安全芯片和指纹模块
3. 功耗低（< 100mA），DCDC 效率优势不明显
4. 设计复杂度高

### 11.2 替代方案二：电池供电

**方案描述**：
- 供电方式：锂电池 3.7V
- 降压方案：LDO 降压至 3.3V
- 充电管理：TP4056 或 MCP73831
- 移动使用

**对比分析**：

| 项目 | 电池供电 | USB 供电 |
|------|----------|----------|
| **移动使用** | 支持 | 不支持 |
| **成本** | $4.5-7.5 | $0.6-1.1 |
| **设计复杂度** | 高 | 低 |
| **维护成本** | 需充电维护 | 无需维护 |
| **体积** | 大（电池） | 小 |

**未选择原因**：
1. 成本高（$4.5-7.5 vs $0.6-1.1）
2. 设计复杂度高（充电管理）
3. 用户需维护电池（充电）
4. 体积大（电池占用空间）
5. FIDO2 安全密钥主要用于 PC 连接，移动需求低

### 11.3 替代方案三：双供电（USB + 电池）

**方案描述**：
- 供电方式：USB VBUS + 锂电池
- 电源切换：自动切换
- 充电管理：TP4056 或 BQ24074
- 灵活使用

**对比分析**：

| 项目 | 双供电 | USB 供电 |
|------|----------|----------|
| **灵活性** | 最高 | 最低 |
| **成本** | $5-10 | $0.6-1.1 |
| **设计复杂度** | 最高 | 最低 |
| **维护成本** | 需充电维护 | 无需维护 |
| **体积** | 最大 | 最小 |

**未选择原因**：
1. 成本最高（$5-10 vs $0.6-1.1）
2. 设计复杂度最高（电源切换、充电管理）
3. 用户需维护电池
4. DIY 项目定位低成本

### 11.4 替代方案四：XC6206P332 低静态电流 LDO

**方案描述**：
- 供电方式：USB VBUS 5V
- 降压方案：XC6206P332 LDO
- 特点：静态电流 < 1μA

**对比分析**：

| 项目 | XC6206P332 | AMS1117-3.3 |
|------|------------|--------------|
| **静态电流** | < 1μA（极低） | ~10mA |
| **输出电流** | 200mA | 1A |
| **封装** | SOT-23（小） | SOT-223（大） |
| **成本** | $0.5-1 | $0.5-1 |
| **成熟度** | 较新 | 成熟 |

**未选择原因**：
1. 输出电流较小（200mA），峰值可能不足
2. AMS1117 更成熟稳定
3. 静态电流差异对 USB 供电影响小

**备注**：可作为备选方案，用于低功耗优化版本。
