# IBio

A DIY FIDO2 key with biometric (fingerprint) verification.

## 项目概述

IBio 是一个开源的 FIDO2 安全密钥项目，旨在通过生物识别（指纹）验证增强身份认证的安全性。该项目允许用户自行制作一个支持 FIDO2/WebAuthn 标准的硬件安全密钥，用于在各种在线服务中进行无密码认证。

### 核心特性

- **FIDO2/WebAuthn 标准**：符合国际认证标准，支持主流浏览器和服务
- **生物识别验证**：集成指纹传感器，提供便捷且安全的身份验证
- **开源硬件**：硬件设计完全开源，用户可自行制作和定制
- **低成本方案**：使用常见的硬件组件，降低制作成本
